# Deploy Preview

Deploy a preview version to Vercel.

## Prerequisites
- Vercel CLI installed: `npm i -g vercel`
- Project linked: `vercel link`

## Deploy Preview
```bash
cd ~/Projects/consulting-site && vercel
```

## Deploy Production
```bash
cd ~/Projects/consulting-site && vercel --prod
```

## Note
Deployment is not yet configured. This command will be updated once Vercel/Netlify is set up.
