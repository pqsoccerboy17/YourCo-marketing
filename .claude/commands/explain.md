# Explain Code

Get a plain-English explanation of any code or file.

## What This Does
Reads and explains code in simple terms that a non-developer can understand.

## Steps
1. **Read the Target**
   - Open the specified file or code section
   - Identify the main purpose

2. **Explain Simply**
   - What does this code do? (one sentence)
   - Why does it exist? (the problem it solves)
   - How does it work? (step by step, no jargon)

3. **Highlight Key Parts**
   - Important functions or sections
   - Any potential issues or "gotchas"
   - Dependencies on other files

4. **Suggest Improvements** (optional)
   - Only if asked
   - Keep suggestions practical

## Usage
```
/explain [filename or code snippet]
/explain src/auth/login.ts
/explain "what does this function do?"
```

## Output Format
- Use analogies and real-world comparisons
- Avoid technical jargon unless necessary
- If jargon is needed, define it
- Keep explanations concise but complete
