# Start Development Server

Start the local development server and provide the URL.

```bash
cd ~/Projects/consulting-site && npm run dev
```

The site will be available at http://localhost:5173

## What to check:
- Light mode colors (terracotta accent)
- Dark mode (toggle system preference)
- Responsive layouts (resize browser)
- Navigation links work
- Contact form displays
